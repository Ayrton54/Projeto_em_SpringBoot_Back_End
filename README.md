## Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- **Java**: A linguagem de programação principal do projeto, utilizada para escrever a lógica do aplicativo.
  
- **Spring Boot**: Um framework de desenvolvimento que simplifica a criação de aplicativos Java baseados em Spring, fornecendo um ambiente de configuração e desenvolvimento mais fácil.

- **MySQL**: O sistema de gerenciamento de banco de dados relacional utilizado para armazenar e gerenciar os dados do aplicativo.

## Como Começar

Para executar este projeto em sua máquina local, siga as instruções abaixo:

1. **Pré-requisitos**: Certifique-se de ter o Java, o Spring Boot e o MySQL instalados em sua máquina.

2. **Clone este repositório**: Use o comando `git clone` para clonar este repositório em seu ambiente local.

3. **Configuração do Banco de Dados**: Configure as informações de conexão com o MySQL no arquivo `application.properties`. Certifique-se de alterar as propriedades do banco de dados conforme necessário.


4. **Executar o Aplicativo**: Utilize o comando apropriado para iniciar o aplicativo Spring Boot.

5. **Acesso à Aplicação**: Após iniciar o aplicativo, você poderá acessá-lo em [http://localhost:8080](http://localhost:8080) no seu navegador.
